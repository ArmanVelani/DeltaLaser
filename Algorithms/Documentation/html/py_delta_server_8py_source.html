3 ## @package pyDeltaServer
4 # A server script written in Python that sends GCode commands
5 # to a delta robot.
6 #
7 # The commands sent to the robot are determined by selections made on an
8 # Android or Web-Based client. The delta bot will have G-code G-code
9 # compatible firmware installed on its control board. This script uses
10 # Printrun libraries provided by kliment on GitHub.
11 #
12 # The server begins by waiting for a client. When a client connects, the
13 # server will attempt to connect to a printer on /dev/ttyACM0. If this
14 # fails, it will attempt to connect to a printer on /dev/ttyACM1; it is
15 # possible that the device name will change from either one of these. If
16 # neither one of these attempts succeed, then the server will begin with
17 # no printer connection. This is especially useful for debugging client
18 # to server communication without having to connect to a control board.
19 #
20 # Author: Dave Mariano
21 # Date: Feb 1, 2015
22 # Filename: pyDeltaServer.py
23 # Github: github.com/d-mariano/DeltaLaser
